## 깃&헤로쿠 등록
#헤로쿠 사이트 setting -> config Vars
BOT_TOKEN : 봇토큰값(5단계에서 저장한값)
GIT_REPO_RESTART : 첫 번째 생성한 repository (ex. rkskekfkfe/repository1)
GIT_REPO : 두 번째 생성한 repository (ex. rkskekfkfe/repository2)
GIT_TOKEN : github 토큰값(13단계에서 저장한값)
